# E-Commerce Compatibility Testing

## Project Overview

This project focuses on compatibility testing of an e-commerce website to ensure consistent functionality and user experience across different browsers and devices.

## Testing Scope

* Homepage Testing
* Login Page Testing
* Product Page Testing
* Navigation Testing
* Broken Link Verification
* Responsive Design Testing

## Browsers Tested

* Google Chrome
* Microsoft Edge
* Brave Browser

## Devices Tested

* Desktop
* Tablet
* Mobile

## Testing Type

* Compatibility Testing
* Cross-Browser Testing
* Responsive Testing
* Manual Testing

## Objective

The objective of this project was to verify that the website functions correctly across multiple browsers and screen sizes while maintaining a consistent user experience.

## Skills Used

* Manual Testing
* Compatibility Testing
* Cross-Browser Testing
* Quality Assurance
* Software Testing
